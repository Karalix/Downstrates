Downstrates

Downstrates is a collaborative editor enabling you to edit and share page written using markdown.

To create and edit a new or existing document go to an URL like this one : localhost:8080/edit/ edit-me you can modify the URL at the left and click the link to see it in action !

You can share an auto-updating document by copying links like this : localhost:8080/read/ edit-me you can modify the URL at the left and click the link to see it in action !
