Downstrates

Downstrates is a collaborative editor enabling you to edit and share page written using markdown.

To create and edit a new or existing document go to an URL like this one : hostname/edit/ editme you can modify the URL at the left and click the link to see it in action !

You can share an auto-updating document by copying links like this : hostname/read/ editme you can modify the URL at the left and click the link to see it in action !

You can share a static version of a document by copying links like this : hostname/snap/ editme you can modify the URL at the left and click the link to see it in action !

There is also the possibility to add your own widget to the editor, to do so, click the "Add Widget" button and copy the URL of the script containing the widget you want to add. You can try to add "/static/em_button.js"

Document names should not contain hyphens or dots in order to be able to get snaps of it.
